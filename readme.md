# 關於CryptoZombies
這是由Loom無償提供的Solidity程式語言教學，藉由編寫一個殭屍遊戲來學習如果開發solidity智能合約。
https://cryptozombies.io/zh/course

但是目前Loom重心轉往其他領域，本課程預計不會再更新(更多資訊可以訪問右上角的"加入群聊"，加入後你會從群組留下的訊息了解更多)前陣子他們更新了Advance Solidity的部分，但是已經有一陣子沒有中文翻譯了。
有鑑於此，我想在我自己學習Solidity的同時，協助他們進行中文翻譯。

目前的翻譯只是我邊學習之餘進行的翻譯，可以說是一個"能看得懂"但不夠漂亮通順、沒有經修排版的版本，而我對Js了解程度也還不夠深(前陣子看到Javascipt大全那一千多頁的書，我就知道我還很遠...)。如果剛好有人在學習CryptoZombies，希望我的翻譯能幫得上忙 :)

不過我在翻譯之餘也發現，之前閱讀原文漏掉的一些細節都補上了，原本只是照著邏輯做，現在更能瞭解其背後的原理。對我自己幫助也很多，所以我推薦你只把這份翻譯當作是輔助。

這份翻譯目前是0.1版本的，偶爾我會習慣用中文，像是物件(object)、函式(function)、實例(instance)，偶爾我會習慣直接用英文稱呼他們，像是 call、test、run、function等等。其實我自己也是習慣東西都要整齊乾淨的強迫症患整，但畢竟我目前式邊學邊翻譯得所以...目前先堪用就好，抱歉了我的強迫症。
在每一章節中，我都會把一些英文單字放在下面，當作我自己複習之用，如果你是搭配原文閱讀，也方便你看一些冷門用字。

祝你測試順利！
